**#Calculadora de Datas de Vencimento**

Aplicação web desenvolvida em **HTML, CSS e JavaScript** para cálculo automático de **datas de vencimento de notas fiscais**, respeitando prazos configuráveis e **limite máximo de valor por dia**.

O sistema foi criado para resolver um problema real de organização financeira, permitindo importar dados diretamente do Excel e exportar os resultados novamente.


**#Funcionalidades**

- Definição de **planos de pagamento** (ex: 21/28/35 ou 28/35/42 dias)
- Configuração de **valor máximo permitido por dia**
- Importação de dados via **colar do Excel**
- Cálculo automático de vencimentos com redistribuição de valores
- Visualização detalhada:
  - Lançamentos individuais
  - Totais por data de vencimento
- **Exportação para Excel (CSV)** com codificação compatível (UTF-8)
